Overview

This repository contains the projects completed as part of the CodeAlpha Cloud Computing Internship Program. These projects demonstrate practical applications of cloud computing concepts, web development, database management, security, and artificial intelligence.

The primary objective of these projects is to develop cloud-based solutions that are scalable, secure, and user-friendly.

---

Projects Included

1. Data Redundancy Removal System

Description

A cloud-based application that identifies and prevents duplicate data records from being stored in a database.

Features

- Duplicate record detection
- Unique data validation
- Database optimization
- Improved storage efficiency
- Secure data management

Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS

---

Installation

Clone Repository

git clone https://github.com/your-username/codealpha-cloud-computing.git

Navigate to Project Directory

cd codealpha-cloud-computing

Install Dependencies

pip install flask
pip install transformers
pip install torch

---

Running the Projects

Create Database

python create_db.py

Start Application

python app.py

Access Application

http://localhost:5000
